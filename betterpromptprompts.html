Step 1 – Concept Definition & Platform Vision

You are an expert AI product strategist. Define the concept and purpose of an “AI Prompt Optimization Platform” — a web app that allows users to input raw prompts and receive optimized versions for different AI models (ChatGPT, Claude, Gemini, Grok, etc.).
Include:
- Target users (beginners, professionals, developers, marketers, etc.)
- Main goals and value proposition
- Key differentiating features from other AI tools
- User flow overview (from input to optimized output)

Step 2 – Branding & Naming

You are a creative branding AI. Propose 5 name ideas, slogans, and branding identities for the AI Prompt Optimization Platform.
Include a color palette, typography style, tone of communication, and emotional appeal for the brand.
Output as a short creative branding guide.

Step 3 – Feature Breakdown & User Flow

As a UX architect, list all core and advanced features for the AI Prompt Optimization Platform.
Organize them into:
- MVP (minimum viable product) features
- Advanced/Pro features
Then, diagram (in text form) the user flow from landing page → prompt input → model selection → optimization → feedback.

Step 4 – UI/UX Wireframe Design

You are a UX/UI designer. Create a wireframe concept (in textual or markdown form) for the platform interface.
Include:
- Input form area for prompt text
- Model selection interface
- Optimized prompt result section
- Feedback/rating module
- Navigation and settings menu
Make it responsive and intuitive.

Step 5 – Frontend Design (HTML + Tailwind CSS)

You are a senior frontend developer. Build the homepage and main interface for the AI Prompt Optimization Platform using HTML and Tailwind CSS.
Include the following sections:
- Header with logo + navigation
- Prompt input area
- Model selection buttons/dropdown
- Result display box
- Feedback section
Ensure responsive design and modern UI aesthetics.

Step 6 – Backend Structure Planning

You are a backend architect. Plan the backend for the platform that processes user input, optimizes prompts, and interfaces with AI APIs.
Include:
- Recommended stack (Node.js, Flask, Django, etc.)
- Data flow diagram (text format)
- API endpoints (input prompt, optimization, model results)
- Security and scalability considerations.

Step 7 – AI Model Integration Plan

You are an AI integration engineer. Create a plan to connect these models:
- ChatGPT (OpenAI API)
- Claude (Anthropic API)
- Grok (xAI API)
- Gemini (Google API)
For each, describe:
- How to authenticate
- How to send prompt requests
- How to handle responses
- API call examples in Python or Node.js

Step 8 – Prompt Optimization Algorithm

You are an NLP prompt engineer. Design an algorithm that takes a user’s raw prompt and optimizes it for each target AI model.
The system should consider:
- Model’s tone (creative, factual, conversational)
- Desired goal (e.g., summary, idea generation, translation)
- Clarity and structure
Output the algorithm’s logic and pseudocode.

Step 9 – Database & User Data Management

You are a database engineer. Create a database schema for the AI Prompt Optimization Platform.
Include tables for:
- Users (accounts, history)
- Prompts (raw, optimized, results)
- Feedback (ratings, comments)
Suggest whether to use SQL or NoSQL and explain why.

Step 10 – Testing and QA

You are a QA tester. Create a comprehensive testing plan for this platform.
Include:
- Unit testing (frontend + backend)
- API reliability testing
- AI output validation tests (prompt quality checks)
- User feedback testing plan
Include example test cases.

Step 11 – Deployment Setup

You are a DevOps engineer. Create a deployment plan for the platform using cloud hosting (AWS, Render, or Heroku).
Include:
- CI/CD pipeline configuration
- Environment variable management for API keys
- Scalability setup
- Backup and monitoring.

Step 12 – Analytics & Feedback System

You are a product analytics specialist. Design a feedback and analytics system for the AI Prompt Optimization Platform.
Include:
- Metrics to track (prompt ratings, usage frequency, model preference)
- Tools to integrate (Google Analytics, Mixpanel, custom dashboard)
- Data visualization ideas.

Step 13 – AI Assistant Personalization

You are an AI personalization engineer. Suggest a system where the platform learns user preferences over time and tailors prompt optimization automatically.
Include how to save personalization data securely and adjust optimization based on feedback loops.

Step 14 – Marketing & Launch Plan

You are a digital marketing strategist. Create a launch and marketing plan for the AI Prompt Optimization Platform.
Include:
- Pre-launch teaser campaign
- SEO and content strategy
- Social media outreach
- Partnerships or influencer marketing ideas
- Conversion funnel strategy.

Step 15 – Monetization & Scaling

You are a startup strategist. Develop monetization ideas for the AI Prompt Optimization Platform.
Include:
- Subscription tiers (Free, Pro, Enterprise)
- Pay-per-API-call model
- Affiliate or plugin ecosystem ideas
